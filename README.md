# 우아한 테크코스 농구게임(프리코스)

숫자야구게임 - 1부터 9까지 서로 다른 수로 이루어진 3자리의 수를 입력을 통해 맞히는 게임

## 결과화면(스크린샷)

## 기능구현

- 1-9 사이의 서로 다른 난수를 생성한다.(O) 

- 사용자 입력.(O)
    - 게임 진행 및 종료 기능.(O)
    - 숫자 입력 기능(난수를 맞히기 위한).(O)
    
- 검증
    - 사용자 입력과 난수를 비교하는 기능.(O)
